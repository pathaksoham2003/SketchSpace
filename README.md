# 🧑‍💻 Collaborative Whiteboard App

A full-stack collaborative whiteboard app with:

- 🧠 **Backend**: Node.js + TypeScript  
- 🎨 **Frontend**: React + TypeScript  
- 🛡️ **Authentication**: Keycloak (Dockerized)  
- 🫢 **Database**: MongoDB (Dockerized)
